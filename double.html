Détection des cicatrices de brûlures de la région du lac Saint-Jean

Une analyse géospatiale réalisé a partir du modèle Prithvi-100M

Ce projet utilise le modèle Prithvi-100M pour détecter les cicatrices de brûlures dans des images satellitaires HLS. Exploitant les capacités avancées de Vision Transformer temporelle, il analyse avec précision les zones touchées par les feux de forêt et propose des visualisations interactives, offrant un outil puissant pour la gestion environnementale et l’analyse géospatiale.

Description du modèle:

La détection des cicatrices de brûlures par Prithvi est un transformateur de vision temporelle, premier du genre, qui a été entraîné par l'équipe d'IBM et de la NASA sur les données Landsat Sentinel 2 (HLS) harmonisées de la partie continentale des États-Unis. Cette démonstration montre comment le modèle a été affiné pour détecter les cicatrices de brûlures.

L'utilisateur doit fournir une image HLS geotiff, comprenant les canaux suivants en unités de réflectance (par exemple 0-1) : Bleu, Vert, Rouge, Narrow NIR, SWIR, SWIR 2.

Dans le cas présent, les données utilisées furent extraites à partir de USGS EarthExplorer.
